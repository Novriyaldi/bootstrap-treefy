Options

Name Type Default Description
expanderTemplate string '<span class="treetable-expander"></span>' HTML Element when you click on that will be collapse/expand branches
indentTemplate string '<span class="treetable-indent"></span>' HTML Element that will be placed as padding, depending on the depth of nesting node
expanderExpandedClass string 'fa fa-angle-down' Class using for expander element when it expanded
expanderCollapsedClass string 'fa fa-angle-right' Class using for expander element when it collapsed
treeColumn number 0 Index of the column which expander should be added to. 0-based.
